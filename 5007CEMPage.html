Software Development Module

September 2021 - December 2021

DESCRIPTION

This module required me to create a website based on a given topic. The focus of this module being creating a REST API using a SQL database to manage the data and using version control correctly and effeciently. The topic given was to create a Frequently Asked Questions website where students can submit questions regarding university topics.

TECHNOLOGY USED

  • HTML
  • CSS
  • Deno
  • Github
  • REST API
  • Single Page Application (SPA)
  • Heroku Deployment
  • My SQL

Project features:

  • This project required me to create a full stack Frequently Asked Questions website where users can create and answer queries using JavaScript, CSS, HTML and a Deno stack.
  • - Version Control: I had used GitHub to assist with feature management during the development of this website. I also used linters to ensure code quality and enforcing checks before commits using some GitHooks
  • - Server-side Scripting: I creeated a database to manage the website data and created a REST API in order to handle the data, including creating, retrieving, updating and deleting the data (CRUD).
  • - Client-Side programming: Finally, I created a responsive user-interface for the website, which allows users to view questions, and add their own, through most devices.

The core functionality consists of:

  • - A homepage where users don't need to be logged in to view the all the questions, however if they are logged in, there should be an Add question button or link that takes users to a screen where they can ask their own questions.
  • - A login and register Page.
  • - An add question page, where when the user is logged in, they can submit their own question to the website.
  • - A Details page which displays the information associated with the question selected. Including the name of the user who posted, time and date.
