Portfolio project

Feb 2022 - March 2022

DESCRIPTION

This project was completed for Wiley Edge. This project me to create a portfolio which would display information about recent projects I have worked on with Wiley and any other technical projects. This is a static website which is being hosted using Netlify.

TECHNOLOGY USED / METHODOLOGIES USED

  • HTML
  • CSS
  • GitHub / Version Control
  • Visual Studio Code

GITHUB LINK

Metro App GitHub

Before starting this project, I created a wireframe to map out the pages I would need for the website. I also included the design of each page, so I could see which elements I needed and where they would go. Doing this allowed me to ensure that all the requirements would be met with my design.

To create this website, I used Visual Studio Code in order to create the code and see the live changes being made.

This website has four main pages which are: Homepage, About, Portfolio and each indivdual project page. The homepage displays an image of me and text which display my name. Upon hovering over a letter of my name, the letter changes color. This was an effect I created in order to add extra interactivity to the website.

The portfolio link on the homepage, has a jump to id link, which means that the page scrolls to the portfolio projects ID which is on the same page. On any other page of the website, the user is taken to the portfolio page which is a seperate page, where the user can view the same projects.
